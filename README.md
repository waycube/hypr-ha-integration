# Hyprland Control (Home Assistant)

Custom Home Assistant integration to monitor and control a Hyprland desktop.

## Install with HACS (Custom Repository)

1. Open HACS in Home Assistant.
2. Go to Integrations.
3. Open the menu (⋮) and choose **Custom repositories**.
4. Add this repository URL.
5. Category: **Integration**.
6. Click **Add** and install **Hyprland Control**.
7. Restart Home Assistant.
8. Add the integration from **Settings → Devices & services → Add integration**.

## Configuration

During setup, provide:
- Name
- Host
- Port (default: 8129)
